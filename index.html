Kubernetes Ingress DNS

A dead-simple DNS Server for home labs. Makes MetalLB load balancers and ingresses just work..

Helm Chart Installation

To install the chart, run the following:

helm repo add kubernetes-ingress-dns https://arkady-emelyanov.github.io/kubernetes-ingress-dns/
helm repo update
helm install ingress-dns kubernetes-ingress-dns/kubernetes-ingress-dns
